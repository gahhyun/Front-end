1. 구성 

          스타일(CSS)의 종류(Property)
            ----
   - 선택자 {속성:값;}
     -----
     스타일(대상)을 적용할 대상(Selector)


2. CSS 선언 방식
    -내장 방식 
        -<style></style>의 내용(Contents)으로 
    -링크 방식
        -<link/>로 외부 CSS 문서를 가져와서 연결하는 방식
    -인라인 방식
        -요소의 style 속성에 직접 스타일을 작성하는 방식
        -선택자 없음
    -@import 방식
        -@import 규칙으로 CSS 문서 안에서 또다른 CSS 문서를 가져와 연결하는 방식


3. CSS 선택자
    1) 기본 선택자
    2) 복합 선택자
    3) 가상 클래스
    4) 가상 요소
    5) 속성 선택자


4. 기본 선택자
    1) 전체 선택자 (Universal Selector)
        - *
        -모든 요소를 선택
    2) 태그 선택자 (type Selector)
        - ABC
        -태그 이름이 ABC인 요소 선택
    3) 클래스 선택자 (class Selector)
        - .ABC
        -HTML class 속성의 값이 ABC인 요소 선택
    4) 아이디 선택자 (ID Selector)
        - #ABC
        - HTML id 속성의 값이 ABC인 요소 선택


5. 복합 선택자
    1) 일치 선택자 (Basic Combinator)
        -ABCXYZ
        -선택자 ABC와 XYZ를 동시에 만족하는 요소 선택
    2) 자식 선택자 (Child Combinator)
        -ABC>XYZ 
        -선택자 ABC의 자식 요소 XYZ 선택
    3) 하위(후손)선택자 (Descendent Combinator)
        -  ABC XYZ
        -선택자 ABC의 하위 요소 XYZ 선택
        -'띄어쓰기'가 선택자의 기호
    4) 인접 형제 선택자 (Adjacent Sibling Combinator)
        - ABC + XYZ
        -선택자 ABC 다음 형제 요소 XYZ 하나를 선택
    5) 일반 형태 선택자 (General Sibling Combinator)
        - ABC ~ XYZ


6. 가상 클래스 선택자 (PShadow-Class)
    1) HOVER
        - ABC:hover 
        -선택자 ABC 요소에 마우스 커서가 올라가 있는 동안 선택
                          ---------------------------
    2) ACTIVE
        - ABC:active 
        -선택자 ABC 요소에 마우스를 클릭하고 있는 동안 선택 
                         --------------------------
    3) FOCUS
        - ABC:focus 
        -선택자 ABC 요소에 포커스되면 선택
                          ---------
    4) FIRST-Child
        - ABC:first-child 
        -선택자 ABC가 형제 요소 중 첫째라면 선택   
                                 
    5) LAST-Child
        -ABC:last-child
        -선택자 ABC가 형제 요소 중 막내라면 선택

    6) NTH-CHILD 
        - ABC:anh-child(n)
        -선택자 ABC가 형제 요소 중 (n)번째라면 선택

    7) NTH-CHILD 
        - ABC:anh-child(2n)  //2의 배수로 출력
        -선택자 ABC가 형제 요소 중 (2n)번째라면 선택
        -n은 0부터 시작(Zero-Based Numbering)

    8) NTH-CHILD 
        - ABC:anh-child(2n+1)  //홀수로 출력
        -선택자 ABC가 형제 요소 중 (2n)번째라면 선택
        -n은 0부터 시작(Zero-Based Numbering)

    9) NTH-CHILD 
        - ABC:anh-child(n+2)  //
        -선택자 ABC가 형제 요소 중 (2n)번째라면 선택
        -n은 0부터 시작(Zero-Based Numbering)

    10) 부정 선택자 (NOT)
        - ABC:not(XYZ)
        -선택자 XYZ가 아닌 ABC 요소 선택


7. 가상 요소 선택자 (Pseudo-Elements)
    1) ABC::=before
          ---------
          인라인(글자)요소
          -선택자ABC 요소의 내부 앞에 내용(content)을 삽입
    
    2) ABC::=after
          ---------
          인라인(글자)요소
          -선택자ABC 요소의 내부 뒤에 내용(content)을 삽입

8. 속성 선택자 (Attribute)
    1) [ABC]
        -속성 ABC를 포함한 요소 선택

    2) [ABC="XYZ"]
        -속성 ABC를 포함하고 값이 XYZ인 요소 선택

9. 스타일 상속
    1) 상속되는 CSS 속성들
        -모두 글자/문자 관련 속성들
        -모든 글자/문자 속성은 아님

         font-style(글자 기울기)
         font-weight(글자 두께)
         font-size(글자 크기)
         line-heigth(줄 높이)
         font-family(폰트 서체)
         text-aling(정렬)

10. 선택자 우선순위
    1) 우선순위
        -같은 요소가 여러 선언의 대상이 된 경우 어떤 선언의 css속성을 우선 적용할지 결정하는 방법
        -점수가 높은 선언이 우선함 
            -점수
                -!important        : 무한대
                -인라인 선언(style) : 1000점  
                -ID 선택자          : 100점
                -Class 선택자       : 10점
                -태그(요소) 선택자        : 1점
                -전체 선택자        : 0점
                -상속은 점수가 없다
        -점수가 같으면, 가장 마지막에 해석된 선언이 우선함

    2) .list li.item {color : red;}
        => 10점 + 1점 + 10점 = 21점 (.=class선택자)

        .list li:hover {color : red;}
        => 10점 + 1점 + 10점 = 21점 (hover은 가상class선택자 이므로 10점)

        .box::before {color : red;}
        => 10점 + 1점 = 11점

        #submit span {color : red;}
        => 100점 + 1점 = 101점
